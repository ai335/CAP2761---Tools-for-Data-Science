**Course Description:**

This course focuses on data manipulation, curation, visualization, exploration, interpretation and modeling using standard packages and tools employed in the field of data science, as well as best practices for maintaining data and software using version control.<br/><br/>

**Prerequisites:**

Students must complete COP1000 with a Grade of C or higher. Students may take COP1000 at the same time with departmental approval.<br/><br/>    

**Course Learning Outcomes:**

**CLO#1:** Apply software packages for data science using programming languages that can include R, Python, SQL and others.

**CLO#2:** Identify patterns and draw conclusions from visualized data.

**CLO#3:** Describe data acquisition, preprocessing, and transformation.<br/><br/>

## Modules
Module 1 - Introduction: What is Data Science?<br/>
Module 2 - Tools: Python, IPython, and Jupyter notebooks<br/>
Module 3 - Tools: NumPy, Matplotlib, and Pandas<br/>
Module 4 - Visualization techniques and exploratory data analysis (EDA)<br/>
Module 5 - Basic Statistical Techniques (and their use in Data Science)<br/>
Module 6 - Statistical hypothesis testing, p-values, p-hacking, and Bayesian inference<br/>
Module 7 - Introduction to Machine Learning<br/>
Module 8 - Unsupervised machine learning<br/>
Module 9 - Supervised machine learning<br/>
Module 10 - Introduction to Deep Learning<br/>
Module 11 - Data Science beyond the code: ethical and legal considerations
